# 📱 GST Calculator App

A simple, fast, and modern GST Calculator Android application built to calculate GST rates instantly with an easy-to-use interface.

---

## 🚀 Features

✅ Add GST Calculation  
✅ Remove GST Calculation  
✅ Supports GST Slabs:
- 3%
- 5%
- 12%
- 18%
- 28%

✅ Instant Calculation Results

✅ CGST + SGST Breakdown

✅ Clean & User-Friendly UI

✅ Lightweight & Fast Performance

✅ Mobile Friendly Design
